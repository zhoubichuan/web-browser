# 四.dom properties

- attribute 是 dom 元素在文档中作为 html 标签拥有的属性；
- property 是 dom 元素在 js 中作为对象拥有的属性。
- 对于 html 的标准属性来说，attribute 和 property 是同步的，是会自动更新的
- 但是对于自定义的属性来说，他们是不同步的.(自定义属性不会自动添加到 property)
